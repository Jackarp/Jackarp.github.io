|	<center>투자의 때를 알려주는 신호들</center>	|
| :-------------------------------------------	|
| 1. 부동산 불황기가 3년 이상 지났다. (지칠데로 지친 매도자들) 	|
| 2. 미분양 물량이 사상 최대로 증가한다.		 	|
| 3. 분양 물량이 역대 최저가 된다.		 	|
| 4. 경매 낙찰가율이 사상 최저 수준이 된다.		 	|
| 5. 전세가율이 사상 최고치에 근접한다.		 	|
| 6. 폐업하는 중개업소가 속출한다.			 	|
| 7. 정부의 부동산 활성화 대책이 등장한다.		 	|
